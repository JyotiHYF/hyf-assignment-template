# Reflection

## 1. Identify and share 3 new things you learned from AI in the above task.

I learned how to generate a random hex color using JavaScript and apply it to the background dynamically. I also learned how to use Google Fonts through CSS @import. Additionally, I understood better how media queries work to make a page responsive for different screen sizes.

## 2. Where did you have to tweak or correct Copilot’s suggestions to suit your needs?

I had to adjust some styling suggestions because the spacing and alignment were not exactly how I wanted them. I also simplified some JavaScript code to better understand what it was doing. In some cases, I renamed variables to make them clearer for myself.

## 3. How would you explain the difference between using Copilot to generate code for you vs. using it as an effective learning partner?

If I only let Copilot generate code, I might copy and paste without fully understanding it. When I use it as a learning partner, I ask why something works and try to modify it myself. The second approach helps me build real understanding and confidence.

## 4. Identify 3 risks of relying too much on AI tools when learning at HackYourFuture.

One risk is not developing problem-solving skills independently. Another risk is copying code without understanding it. A third risk is becoming too dependent on AI instead of learning how to debug and research solutions myself.
